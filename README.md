# 🏛️ AI Lawyer - Indian Constitution Legal Assistant

An AI-powered legal consultation application providing intelligent guidance based on the Indian Constitution.

## 🌟 Features

- **AI-Powered Consultation**: Ask legal questions in natural language and receive intelligent, contextual answers.
- **Comprehensive Database**: Access over 179+ articles across all parts of the Indian Constitution.
- **Smart Search**: Find articles by number, title, or keywords.
- **Plug & Play**: Works both as a Python Flask app AND as a static site (GitHub Pages).
- **Premium UI**: Modern, professional "Glassmorphism" interface.

## 🚀 How to Use

### Option 1: Just open the file (Static/GitHub Pages)
1. **Double-click `index.html`** in your browser.
2. That's it! Everything (search, AI logic, and data) runs directly in your browser.

### Option 2: Run with Flask (Backend)
1. Install dependencies: `pip install -r requirements.txt`
2. Run the server: `python app.py`
3. Open `http://localhost:5000`

## 📁 Project Structure

- `index.html` - The main application page.
- `static/` - Contains CSS, JavaScript, and the Constitution database.
- `app.py` - Flask backend (optional).
- `CONSTITUTION_COVERAGE.md` - Detailed list of covered articles.
- `requirements.txt` - Python dependencies for Flask.
